## Installation
<StepSection>
    <Step $row>
        <StepL>
            ### <StepNum />Create a project
            If you don't have a Blazor project, create one first. It's recommended to refer to [Getting Started - Blazor](https://blazorjs.dev/guide/)
        </StepL>
        <StepR>
            <CodeTabs>
                {[
                    {
                        name: 'Terminal',
                        lang: 'bash',
                        children: `
                            npm create blazor@latest project -- --template vanilla-ts
                            cd project
                            npm install
                        `
                    }
                ]}
            </CodeTabs>
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Initialize configuration file
            Run `npm create @master/css@rc` to create a configuration file [master.css.ts](/guide/configuration).
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm create @master/css@rc
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Install Master CSS Extractor
            Install the Master CSS extractor into your project via package managers.
        </StepL>
        <StepR>
            ```install
            **@master/css-extractor.blazor@rc**
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Set up CSS extractor
            - Create a `vite.config.ts` file
            - Add a `masterCSSExtractor` to the vite `plugins`
        </StepL>
        <StepR>
            ```js name=vite.config.ts
            +import { defineConfig } from 'vite'
            +import masterCSSExtractor from '@master/css-extractor.vite'

            +export default defineConfig({
            +    plugins: [
            +        masterCSSExtractor()
            +    ]
            +})
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Import virtual CSS module
            Import the virtual CSS module `.virtual:master.css` into the `main.ts`.
        </StepL>
        <StepR>
            ```ts name=main.ts
            …
            +import '.virtual:master.css';
            ```
        </StepR>
    </Step>
    <Step $row>
        <StepL>
            ### <StepNum />Launch server
            Run the Blazor development server.
        </StepL>
        <StepR>
            ```bash name=Terminal
            npm run dev -- --open
            ```
        </StepR>
    </Step>
    <Step>
        ### <StepNum />Hello world!
        Now style your first element using Master CSS syntax!
        ```html name=index.html
        <!DOCTYPE html>
        <html lang="en" style="display: none">
            …
            <body>
        +        <h1 class="font:40 font:heavy italic m:12x text:center">Hello World</h1>
                …
            </body>
        </html>
        ```
        Open your browser to watch the changes.
        <HelloWorld url="localhost:5173" />
    </Step>
</StepSection>
